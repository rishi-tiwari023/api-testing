# API Testing Project

Simple Express.js API server.

## File Structure
```
api-testing/
├── package.json          # Dependencies & scripts
├── server.js             # Express API server
└── README.md             # This file
```

## Quick Start
```bash
npm install
npm start          # Run server (port 3000)
```

## API Endpoints
- `GET /api/users` - Returns user list
- `GET /api/health` - Health check
